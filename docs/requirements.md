# Requirements

These are what the Stardeck needs before considered complete.
After these are fulfilled, we can always improve it then, but these are our priorities.
Afterwards, we can decorate it, give it a cooler design, etc

## Functional Requirements
- Must boot Linux
- Must run Python
- Must be battery-powered
- Must have keyboard input

## Non-Functional Requirements
- Portable
- Durable enough for backpack use
- 3D printable

## Constraints
- Student budget (can't spend too much on this)
- First cyberdeck (haven't done much hardware work before)
- Limited electronics experience (Arduino starter kit, that's it)

## Success Criteria

Stardeck v0.1 is considered complete when:

- Linux boots successfully
- Python programs can be executed
- Device operates from battery power
- Keyboard input functions correctly
- Device can be transported in a backpack
