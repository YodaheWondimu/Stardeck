# Stardeck

**The Goal:** Build the ugliest functional cyberdeck possible this summer and submit it to Stardance.

Stardeck prioritizes functionality and learning over aesthetics for its first version. Once the system is operational, later iterations will focus on usability, appearance, and additional features. This approach prioritizes reaching a working MVP before focusing on aesthetics or refinement.

## Objectives
- Learn electronics and hardware integration
- Apply CAD to an embedded systems project
- Learn Raspberry Pi development and deployment
- Practice engineering documentation and iterative design
